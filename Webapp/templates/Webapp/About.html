Brief description of the system

As predictive analysis is emerging to set a new future ahead where we can predict the outcomes from the historic data we had an idea of what if we do have the capability of presumptive diagnosis (the one diagnosis earlier) of disease using the actual patient data so, for that in mind we start by first working on the dengue patients data as it is one of the prominent disease in Pakistan.

We collected actual dengue patients data from different sources and align up all together with the best machine learning algorithms not by just using one algorithms we used around four best algorithms in mechanism we refer as “Voting mechanism”. This voting mechanism work in a way that all algorithm run on the same dataset and they produce their own result set which then compare and if match the vote count increase for that prediction, the one with back of most algorithms become the final prediction for the patient.
